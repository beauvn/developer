    Deployment

    Trên Server :¶

    Thông tin Server¶

    Máy chủ : labs.awe7.com

    Tài khoản : liên hệ cấp cao hơn để lấy

    Pass : liên hệ cấp cao hơn để lấy

    Cài đặt Server¶

    $ ssh beau@labs.awe7.com
{Nhập pass - NOTE: pass sẽ không hiển thị}

    Kiểm tra thư mục của mình có tồn tại hay không qua ls:

    $ ls
bichdn vnmilky

    Nếu không xuất hiện thư mục ng dùng của mình, clone thư mục của vnmilky bằng lệnh 

    cp -R vnmilky/ foldername/

    Chạy lệnh như trên local host 

    cd foldernam/name-project
npm install
composer install 
gulp

    Mở trình duyệt với địa chỉ : http://labs.awe7.com:3000

    (Lưu ý: Thông thường port sẽ là 3000 tuy nhiên nếu nhiều chương trình chạy cùng lúc port sẽ thay đổi, xem thông báo trên cửa sổ CMD/Ternimal để biết chính xác địa chỉ)

    Đồng bộ từ editor lên server¶

    Sử dụng Sublime Text 3, cài đặt plugin: Sublime SFTP

    Thông tin server như trên, đường dẫn thư mục làm việc: /{user}/{project}

    {
"type": "ftp",
    "save_before_upload": true,
    "upload_on_save": true,
    "sync_down_on_open": false,
    "sync_skip_deletes": false,
    "sync_same_age": true,
    "confirm_downloads": false,
    "confirm_sync": false,
    "confirm_overwrite_newer": false,
    "host": "labs.awe7.com",
    "user": "beau",
    "password": "",
    "port": "21",
    "remote_path": "/{user}/{project}",
    "ignore_regexes": [
        "\\.sublime-(project|workspace)", "sftp-config(-alt\\d?)?\\.json",
        "sftp-settings\\.json", "/venv/", "\\.svn/", "\\.hg/", "\\.git/",
        "\\.bzr", "_darcs", "CVS", "\\.DS_Store", "Thumbs\\.db", "desktop\\.ini", "/node_modules/", "/public/"
    ],
    "connect_timeout": 30,
}

    Warning

    Lưu ý cần phải có file .gitignore trong thư mục project

    Nội dung file .gitignore

    node_modules/
.DS_Store
patternlab.json
.sass-cache/*
/sass-cache
Thumbs.db
.idea/
public/
/styleguidekit/.sass-cache
gulpfile.js
sftp-config.json

    Trên Local¶

    Trường hợp không kết nối được đến server :

    Clone Project¶

    Clone project về local và cd đến thư mục project 

    $ cd project/

    Cài Gulp¶

    Đầu tiên tiến hành cái Gulp Global bằng lệnh 

    $ npm install --global gulp-cli

    Sau đó cd vào thư mục project và chạy lệnh 

    $ npm install gulp
$ npm install 
$ composer install
$ gulp

    Vào bằng đường dẫn : http://localhost:3000
